# BetUS Landing Page (Affiliate) – v1.0

Static landing page optimized for US search intent:
- BetUS Review
- Bonuses
- Sports & leagues coverage
- AI Predictions (frontend mock)
- SEO meta + Review/FAQ schema
- Affiliate disclosure + responsible gambling notice

## Setup
1) Replace `YOUR_AFFILIATE_LINK` in:
- index.html
- js/main.js

2) Replace canonical URL `https://YOURDOMAIN.com/` in index.html with your GitHub Pages URL or domain.

## Deploy on GitHub Pages
- Settings → Pages → Deploy from branch → select `main` and `/root`.
